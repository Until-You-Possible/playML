4： 关于一些随机数
    一些基本概念
    * 均值， 方差

所谓均值，是表示一组数据中集中趋势的数量，是指在一组数据中所有数据之和再除以这组数据的个数，它反应的是数据集中趋势的一项指标。
方差：方差是各个数据与平均数之差的平方的和的平均数。
    eg: 有 1、2、3、4、5这组样本，其平均数为（1+2+3+4+5）/5=3，而方差是各个数据分别与其和的平均数之差的平方的和的平均数，则为：
    [(1-3)^2+(2-3)^2+(3-3)^2+(4-3)^2+(5-3)^2]/5=2，方差为2。
